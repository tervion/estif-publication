# ============================================================================
# Notes on version constraints
# ============================================================================
#
# numpy<2.0.0
#   numpy 2.0 introduced breaking API changes. The ESTIF model code uses
#   numpy 1.x conventions throughout. Constraint is intentional.
#   Warning: some unrelated system packages (e.g. opencv) may flag this as
#   incompatible. That warning is harmless — opencv is not used by ESTIF.
#
# scipy<2.0.0
#   Maintains stable API for integrate.quad and optimize.minimize.
#   scipy 1.10–1.17 all work correctly with the constraint as written.
#
# matplotlib<4.0.0
#   Prevents breaking changes to plot/axes API.
#
# astropy<7.0.0
#   Used for FlatLambdaCDM cosmology class and physical constants.
#   astropy 5.x and 6.x both work correctly.
